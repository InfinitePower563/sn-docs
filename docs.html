HttpRequest

HttpRequest is a class that is used to send HTTP requests to a server. You can specify the type of request you want to send, the URL, and the data you want to send.

The RequestType enum is used to specify the request type. 

Syntax:
HttpRequest(RequestType requestType, String url, Map<String, String> parameters);

Perform a request
String performRequest(int requestHeaders, String[] hKey, String[] hVal) throws HttpException;

Perform a request that returns a JSON object:
JSONObject performRequestJson(int requestHeaders, String[] hKey, String[] hVal) throws HttpException;
Getter and setter methods are also included.

RequestType

The RequestType enum is used to specify the request type. 

Syntax:

enum {
    GET,
    POST,
    PUT,
    DELETE,
    HEAD,
    PATCH,
    OPTIONS
}
Refer to above for proper usage.

JsonMapper

The JsonMapper class is used to convert Strings to JSONObjects. 

Syntax:
JSONObject jsonMapper(String jsonString);

HttpException

The HttpException class is used to handle HTTP exceptions. 

Syntax:
HttpException(String msg, Throwable cause);

Catching:
try {
    //...
} catch (HttpException e) {
    //...
}
